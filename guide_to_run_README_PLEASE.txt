Run Offline:
Step A:
Open CMD
cd folder backend
run commnd "npm run dev"

Step B:
Open XAMPP
Toggle the options for Apache and MySQL
open https://localhost/phpmyadmin/index.php?lang=en in any browser
Open SQL tab 
Copy the command in the database.sql
Paste it into the sql tab command console
CLick GO

Step C:
Open folder backend
Open .env file with any text editor or VsCode
Copy the code from Local.env.txt to .env file

Step D:
Open CMD
cd folder project itself
run command "python -m http.server 5500"
open url "http://127.0.0.1:5500/"
